Программа для извлечения ключевых терминов из текста.

Для тестирования используется текстовый файл "tren_text_1.txt".
Так же можно добавлять и другие текстовые файлы с расширением ".txt"

Программа ещё в разрадботке, и некоторый функционал не работает.

UPD. 18.06.18 - Extracting_keywords(0.1.0)
    Выпущенна первая рабочая сборка. Функционал ограничер работой с файлами '.txt'.
    Доступна только одна предметная область. Не признаки отбора реализованны.
    Доступно извлечение ключевых слов. Доступна базовая оценка найденных слов.
